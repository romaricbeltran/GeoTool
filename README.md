# GeoTool

🛠️ Outil de visualisation 3D écrit en C++ avec OpenGL & ImGui.  
📦 Construit avec CMake, compatible MSVC / Windows.

## ✅ Build (Windows / MSVC)

```bash
cmake -S . -B build -G "Visual Studio 17 2022" && cmake --build build --config Release
```

## 🔧 Stack utilisée
- C++20
- OpenGL 3.3 Core
- GLFW / GLAD / ImGui / GLM
- CMake
